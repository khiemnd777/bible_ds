# Bible Decision Simulator

Offline-first, data-driven Flutter narrative game with:

- JSON content engine
- Deterministic daily scheduler with topic-of-week + streak
- Stat engine with clamped values
- Anime portrait resolver
- Reflection and Bible verse flow
- SharedPreferences persistence
- Riverpod architecture
- Editor content preview tooling
- Monetization scaffolding (Rewarded Ad + Donate dialog)

## Run

```bash
flutter pub get
flutter run
```
